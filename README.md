# goti_project
- 액티비티(놀이공원, 워터파크, 박물관 등)을 중계하는 O2O 플랫폼 서비스
- Angular 기반의 하이브리드 웹 앱

## 기능
- ~~PASS권 온라인 결재~~ 해당 기능 잠겨 있음.
- ~~페이스북, 구글 소셜 가입 및 로그인~~ 해당 기능 잠겨 있음.
